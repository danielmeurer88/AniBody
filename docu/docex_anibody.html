Anibody.import()
Let's say we want a ColorPicker. The class name is Anibody.ui.Colorpicker. Writing it everytime you want to instantiate a new ColorPicker is quiet cumbersome.

Therefore Anibody provides an import function:

Anibody.import(Anibody.ui.ColorPicker);
var cp = new ColorPicker();
cp.Start();

If ColorPicker is still too long or the name collides with another class in your project then there is the possibility to use an alias.

Anibody.import(Anibody.ui.ColorPicker, "Copi");
var cp = new Copi();
cp.Start();
